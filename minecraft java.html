Minecraft – Informasi Dasar

Panduan singkat untuk pemula tentang game Minecraft

Apa itu Minecraft?

Minecraft adalah game sandbox (kotak pasir) yang dikembangkan oleh Mojang Studios (sekarang bagian dari Microsoft). Pemain bisa membangun, menjelajah, bertahan hidup, dan berkreasi di dunia yang terbuat dari blok.

Dua versi utama:

  • Java Edition – Versi PC resmi, paling banyak fitur dan komunitas besar
  • Bedrock Edition – Versi cross-platform (mobile, console, Windows 10/11)

Fitur Dasar Minecraft

  • Membangun apa saja menggunakan blok (rumah, kastil, kota, mesin redstone)
  • Menjelajahi dunia yang dihasilkan secara acak (biome, gua, desa, Nether, End)
  • Bertahan hidup: kumpulkan makanan, lawan monster (zombie, creeper, skeleton)
  • Crafting: buat alat, senjata, armor, dan item dari bahan yang dikumpulkan
  • Mode permainan: Survival, Creative, Adventure, Spectator
  • Multiplayer: main bersama teman di dunia yang sama

Block & Item Dasar

Di Minecraft Java Edition (versi terbaru 1.21+), ada sekitar 1,200+ jenis block (termasuk variasi slab, stair, wall, dll.).

Beberapa kategori utama:

  • Building Blocks: Kayu, batu, batu bata, beton, wol, terracotta
  • Natural Blocks: Rumput, tanah, pasir, kerikil, salju
  • Ore & Mineral: Besi, emas, berlian, lapis lazuli, redstone, coal
  • Nether & End: Netherrack, obsidian, end stone, purpur
  • Decoration: Bunga, pagar, tangga, karpet, lukisan

Item dasar: Pickaxe, Sword, Armor, Food (daging, apel, roti), Torch, Chest, Furnace, Crafting Table.

Cara Mulai Bermain

  1. Beli dan download Minecraft resmi di situs Mojang
  2. Login dengan akun Microsoft
  3. Pilih mode Survival atau Creative
  4. Pukul pohon untuk ambil kayu → buat crafting table → mulai bertahan hidup
